# 深柴电力 3D 展示网站 — 项目交接文档

## 项目概述

深柴电力（Shenchai Power）品牌的 3D 交互展示网站。基于 React + Three.js + Vite 构建，通过滚动控制 3D 场景切换，展示公司产品与技术的五个核心模块。

## 技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| React | ^19.2.7 | UI 框架 |
| Three.js | ^0.172.0 | 3D 引擎 |
| @react-three/fiber | ^9.1.0 | React 与 Three.js 桥接 |
| @react-three/drei | ^10.0.0 | Three.js 辅助工具 |
| Vite | ^8.1.1 | 构建工具 |

## 核心功能

### 1. 模块与导航
- 共 5 个模块：深柴动力(home)、ALL-IN-ONE、动力系统稳定输出(engine)、气流设计(airflow)、减震设计结构(chassis)
- 左侧导航栏（StateTable）点击跳转模块
- 滚动控制场景切换（600vh 滚动区域）

### 2. 3D 场景
- 加载 `generator.glb` 3D 模型
- 粒子系统（200个浮动粒子，暖色调）
- 多光源系统（环境光、方向光、点光、半球光）
- OrbitControls（已禁用交互，仅跟随滚动）

### 3. 语言切换
- 中英双语支持（翻译对象 T.zh / T.en）
- 右上角按钮切换

### 4. 交互动画
- 模块切换时内容淡入淡出（CSS transition）
- 标题框缩放/透明度入场动画（titleBoxEnter）
- "500kw发电机" 金色渐变流光文字（shimmer 动画）

### 5. 装饰效果
- 顶部滚动进度条（金色渐变）
- 科技网格背景
- 浮动技术标签（500kW / 50Hz / IP55 / ≥92%）
- 左右两侧装饰竖线
- 底部版权信息
- 每个模块不同颜色的柔和光晕
- 右下角大序号 01/05

## 开发指南

### 本地运行
```bash
cd about-3d
npm install
npm run dev        # 开发服务器（默认 5174 端口）
```

### 生产构建
```bash
npm run build      # 输出到 dist/ 目录
npm run preview    # 预览构建结果
```

### Cloudflare Pages 部署
1. 构建命令：`npm run build`
2. 输出目录：`dist`
3. 无需额外配置，直接上传 dist 目录或连接 Git 仓库

## 翻译系统

在 `App.jsx` 底部的 `T` 对象中添加/修改翻译：
```javascript
const T = {
  zh: { key: "中文文本" },
  en: { key: "English Text" },
};
```
使用方式：`{t("key")}`

## 重要说明

1. **3D 模型文件** `generator.glb` 位于 `public/` 目录
2. **所有模块图片**位于 `public/images/`，替换同名文件即可
3. **Logo 图片**使用 `public/logo.png`（透明背景），CSS 已配置 `brightness(0) invert(1)` 转为白色
4. **模块数据**在 `App.jsx` 的 `STAGES` 数组中定义，含摄像机位置、视角等参数
5. **备份版本**位于 `work/about-3d-BACKUP-20260709-175735`

## 注意事项
- 3D 模型较大（约 3-5MB），建议优化后再部署
- 已有 dist 目录可直接用于部署
- 滚轮滚动控制场景切换，鼠标拖拽已禁用
- 移动端有响应式适配（768px 断点）
